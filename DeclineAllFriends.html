RBR Decline All Friend Requests

Description

This code is useful for when you want to delete all friend requests!

Instructions

Download the zip file at the bottom of the page!

If you haven't already, create a replit account.

Click plus sign on top right to create a new project.

Go to templates tab and then select Python

Once your project is made, on the list of files on top left click on 3 dots next to "Files" row with the 2 icons and 3 dots.

Click import folder and then import your extracted folder

Click green run button and then in console on the right, follow what it says.

Zipped folder with code

Download .zip

Import this code into replit and then press run!
